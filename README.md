The dataset is from https://www.kaggle.com/competitions/titanic

intinya, proyek ini menggunakan dataset titanic dari kaggle untuk menganalisis faktor-faktor yang mempengaruhi keselamatan penumpang pada tragedi tenggelamnya kapal titanic

dataset ini berisi informasi tentang penumpang Titanic, termasuk data demografi, informasi tiket, serta status keselamatan penumpang

Dataset ini memiliki tiga tipe atribut utama:

- nominal: sex, embarked, cabin
- ordinal: pclass (kelas penumpang: 1 = kelas atas, 2 = kelas menengah, 3 = kelas bawah)
- numerik: age, fare, SibSp, parch

tujuan dari proyek ini adalah melakukan eksplorasi data untuk memahami hubungan antara karakteristik penumpang dengan kemungkinan keselamatan koko
